# Mariah Acessórios - Site Final

Projeto estático HTML/CSS/JS do site Mariah Acessórios, pronto para deploy.

### Como usar
- Faça upload do arquivo `mariah-acessorios.zip` ou faça push para um repositório GitHub.
- Deploy recomendado: Vercel (Add New Project → Import → Upload ZIP).
- O botão do quiz redireciona para `https://example.com/quiz`. Substitua no `script.js` pelo link real do quiz quando tiver.

### Estrutura
- index.html
- style.css
- script.js
- logo.png (favicon: favicon.png)
- images/img1.jpeg ... img10.jpeg
